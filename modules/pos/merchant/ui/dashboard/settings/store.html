วิธีสมัครสร้าง LINE Channel
ทำตามขั้นตอน 5 ขั้น ใช้เวลาประมาณ 10 นาที
1
สร้าง LINE Official Account (LINE OA)
ถ้ายังไม่มี LINE OA ให้สมัครก่อนที่ account.line.biz → กด "สร้างบัญชี" แล้วกรอกข้อมูลร้านค้า
2
เปิด LINE Developer Console
ไปที่ developers.line.biz → Login → กด "Create a Provider" → ตั้งชื่อ Provider (เช่น ชื่อธุรกิจของคุณ)
3
สร้าง Channel แบบ Messaging API
ใน Provider ที่สร้าง → กด "Create a channel" → เลือก "Messaging API" → กรอกชื่อ Channel → กด Create
4
คัดลอก Channel Secret และ Access Token
ใน Channel ที่สร้าง → แถบ "Basic settings" → คัดลอก Channel secret | แถบ "Messaging API" → เลื่อนลงล่าง → กด "Issue" เพื่อสร้าง Channel access token แล้วคัดลอก
5
ตั้ง Webhook URL แล้วกด Verify
ใน แถบ "Messaging API" → หัวข้อ Webhook settings → วาง URL ด้านล่างลงไป → เปิด "Use webhook" → กด "Verify" ต้องขึ้น Success
💡
หลัง Verify สำเร็จ → กลับมากรอก OA ID, Token, Secret ในฟอร์มด้านบน → กด บันทึก → ระบบพร้อมส่งบิลผ่าน LINE ได้ทันที!

วิธีหา LINE User ID
ทำตาม 3 วิธีนี้ เลือกแบบที่สะดวก
1
ใช้บอท "LINE User ID Finder" (ง่ายสุด)
1. เปิด LINE → ค้นหา @lineuid หรือ LINE ID Finder
2. กด Add Friend → ส่งข้อความอะไรก็ได้
3. บอทจะตอบกลับ User ID ของคุณทันที รูปแบบ Uxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
2
ดูจาก Webhook Log ของ VIIV
1. ให้เจ้าของร้าน Add Friend LINE OA ของร้านก่อน
2. ส่งข้อความใดๆ ไปที่ LINE OA
3. ระบบ VIIV จะ log User ID ไว้ → ดูได้จาก Server Log หรือรอฟีเจอร์ "ดู User ID" ที่จะมาเร็วๆ นี้
3
Group ID (สำหรับแจ้งทั้งทีม)
1. สร้างกลุ่ม LINE ของทีมงาน
2. เพิ่ม LINE OA เข้ากลุ่ม
3. ส่งข้อความในกลุ่ม → OA จะได้รับ Group ID อัตโนมัติ
4. Group ID รูปแบบ Cxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx (ขึ้นต้นด้วย C)

LINE OA ต้องได้รับสิทธิ์ "เพิ่มเป็นเพื่อน" และยอมรับ Group Chat ก่อน ตั้งได้ที่ LINE OA Manager → Response → Group Chat
